# Grocery Store API

API для магазина продуктов с категориями, подкатегориями, товарами и корзиной пользователя.

## Функционал

- ✅ Управление категориями и подкатегориями (админка)
- ✅ Управление товарами с тремя размерами изображений (админка)
- ✅ Публичные эндпоинты для просмотра категорий и товаров с пагинацией
- ✅ Токен-аутентификация
- ✅ Корзина пользователя (добавление, изменение количества, удаление, очистка)
- ✅ Подсчет общего количества товаров и суммы в корзине
- ✅ Документация Swagger/ReDoc

## Технологии

- Python 3.12
- Django 6.0
- Django REST Framework
- SQLite
- drf-yasg (Swagger)
